# Todo Musica
Un Ecommerce online sobre la venta de instrumentos musicales.

# Descripcion de como fue llevado a cabo el proyecto: 
* Home donde estan todos los productos disponibles
* NavBar con categorias para poder buscar instrumenos en especifio y carrito
* Detalles de cada producto con foto, descripcion, precio y stock
* En los detalles del producto se pueden modificar las cantidades y enviar ese producto al carrito, luego si quiero enviar mas cantidades de un producto al carrito, lo puedo hacer de la misma forma
* Carrito de compras, con la posibilidad de eliminar los productos dentro de el, tanto el que uno desee, como todos y poder finalizar la compra.
* Checkout form, un formulario que el cliente tiene que rellenar para poder realizar la compra, y luego devolverle el id de compra
* Todos los productos provienen de una base de datos en Firebase
* La orden de compra del cliente tambien es enviada a Firebase
